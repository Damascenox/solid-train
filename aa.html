Análise de Salários em Dados: Um Panorama do Brasil e do Mundo

Arthur Damasceno, Inácio Guimarães, João Rodrigues, Vítor

Introdução

O propósito do presente projeto é realizar uma análise estatística descritiva de dados relativos aos trabalhadores do setor de dados ao redor do mundo.

  • Analisar tendências globais e locais.
  • Identificar fatores que influenciam a remuneração.
  • Oferecer um panorama para profissionais da área.

Coleta de Dados

As fontes de dados são provenientes de diferentes bancos de dados do Kaggle:

1. Data Science Salaries (“DSS”)

  • Fonte: Kaggle DSS
  • Descrição: Salários de 2020 a 2024.

2. State of Data Brazil (“SDB”)

  • Fonte: Data Hackers
  • Descrição: Pesquisa anual (2019-2024) com profissionais de dados no Brasil.

Conclusão

A análise revela tendências distintas entre o mercado de dados brasileiro e o global.

  • Disparidade Salarial: Existe uma diferença significativa entre os salários no Brasil (em R\() e no mundo (em US\)), mesmo considerando as conversões.
  • Modalidade de Trabalho: O trabalho remoto e híbrido são predominantes em ambos os cenários, mas com variações entre os cargos.
  • Educação e Salário: Níveis mais altos de escolaridade, como mestrado e doutorado, estão correlacionados com faixas salariais mais elevadas, especialmente no Brasil.
  • Cargos de Destaque: Cargos de gestão e especializados (como Diretores, Head of Data, ML Scientist) lideram os rankings de remuneração tanto no Brasil quanto no mundo.
